In order to build this game most easily, flash builder or the flash builder eclipse plugin is required.  After cloning the repository, select File->Import->Other... and then "Existing Projects into Workspace."  Simply select the main EgyptianInvasion root directory, and Flash Builder should take care of the rest including any dependencies.
